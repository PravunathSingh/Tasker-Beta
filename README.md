# Tasker (Demo) | Managing Your Tasks Never Got Easier!

A next generation task manager built using React. Uses JSON server to create a mock backend to persist the data and the changes made to the app.

Features:

- Add A New Task

- Set Reminder For Added Tasks

- Delete Tasks Once Completed

### This is the beta version of the app and new features will be added soon.
